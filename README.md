# Scoop

- 集成`fastgit`，能在墙内更快的下载来自`github`的文件
- 为了降低`fastgit`的负担，仅在墙内生效，判断依据能否访问`v2ray.com/robots.txt`
- 安装脚本改自`ScoopInstaller/Install`

## 安装

```
iwr -useb raw.fastgit.org/star2000/scoop/master/bin/install.ps1 | iex
```

## 用法

```
scoop help
```
